# Stan Playground Documentation

Stan Playground is a browser-based editor and runtime environment for Stan models. Users can edit, compile, and run models, as well as analyze the results using built-in plots and statistics or custom analysis code in Python or R, all with no local installation required.

[Visit the live site!](https://stan-playground.flatironinstitute.org)

## Features

- **Browser-based**: No local installation required
- **Interactive**: Edit, compile, and run Stan models in real-time
- **Analysis tools**: Built-in plots and statistics, plus custom Python/R analysis
- **Sharing**: Share your work via GitHub Gists
- **Educational**: Well-suited for teaching and learning Stan
